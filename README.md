[javaDocFramework.html](https://github.com/user-attachments/files/29614712/javaDocFramework.html)# Projeto Framework para Datapacks Minecraft

Projeto em Java para criação automatizada de arquivos JSON para datapacks do Minecraft.

Utilizando o design pattern Facade para gerar arquivos JSON de:

* Receitas
* Loot tables
* Conquistas
* Encantamentos
